# 中間成果物（Reactログインアプリ）

## 概要
- Reactを用いたログイン管理アプリ
- ユーザー認証（3回失敗でロック機能付き）
- ログイン後、艦艇ランダム表示ボタン
- 背景画像切替機能あり

## ファイル説明
- App.js ：メインプログラム
- App.css ：画面デザイン
- users.json ：ユーザー情報管理
- warships.json ：艦艇データ管理
- background.jpg ：背景画像
- 実行画面キャプチャ ：動作確認用スクリーンショット

## 実行手順
1. 任意のReact環境にファイルを配置
2. `npm start`で起動
3. ログインして機能を確認
